# Ecommerce-Customer-Analysis-BI-Dashboard 
본 프로젝트는 샘플 스토어의 데이터를 활용하여 **데이터 전처리, RFM 기반 고객 분류, 그리고 태블로(Tableau)를 이용한 비즈니스 KPI 시각화**를 수행한 통합 분석 프로젝트입니다. 중소 이커머스의 분석 역량 격차를 해소하고, 데이터에 기반한 의사결정을 돕는 것을 목적으로 합니다.

## 주요 사용 기술
- **Language:** Python 3.12.7
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Visualization:** Tableau Public

## 주요 수행 내용
### 1. 데이터 전처리 및 엔지니어링
- 결측치 제거 및 데이터 타입 최적화
- 중복된 행 제거

### 2. RFM 기반 고객 분류 및 분석(Segmentation)
- 고객을 **우수, 일반, 이탈 위험** 세그먼트로 분류.
- 세그먼트별 여러 특징(구매 주기, 카테고리 다양성, 지역적 특이점 등)들을 분석하여 맞춤형 마케팅 전략 제안.

### 3. 비즈니스 KPI 및 스토어 현황 시각화 (Tableau)
- **핵심 지표(KPI) 모니터링:** 전체 매출, 수익, 주문 건수 등을 한눈에 파악할 수 있는 대시보드 구축.
- **지역별 매출 분석:** 어느 도시에서 가장 많은 구매가 일어나는지 지도 시각화로 구현.
- **상품 성과 분석:** 판매량 상위 제품 및 카테고리별 수익성 분석을 통해 재고 최적화 인사이트 도출.
- **기타 분석:** 위의 분석 외에도 월별, 고객 세그먼트별(기업 고객, 홈오피스, 소비자) 등 다양한 방면에서 분석 진행.

### 4. AI 활용 데이터 앱 프로토타입 구현 (Lovable)
분석 결과를 실제 비즈니스 환경에서 어떻게 활용할 수 있을지 제시하기 위해, **AI 네이티브 빌더인 Lovable**을 사용하여 **고객 분석 앱 프로토타입**을 구현하였습니다.
- **비즈니스 솔루션 제안:** RFM 분석으로 도출된 세그먼트(우수, 일반, 이탈 위험)를 마케팅 담당자가 한 눈에 파악할 수 있도록 UI 구현.
- **신규 고객 관리 인터페이스:** 월별 신규 유입 추이와 지역별 데이터를 한눈에 확인하는 대시보드 화면 설계.
- **데이터의 앱화(Application):** 파이썬의 복잡한 분석 결과를 실무자가 즉시 이해하고 행동(Action)으로 옮길 수 있는 인터페이스 디자인.


## 주요 분석 결과
### 신규 고객 유입 및 가치 극대화
- 스토어 오픈 초기 대비 신규 유입이 하향 안정화되는 추세를 확인(2023~2024년 신규 유입이 10명 미만으로 성장 정체기).
- 인사이트: 무분별한 마케팅 비용 지출보다는 유입 경로가 확실한 채널에 집중하고, 기존 고객의 **LTV(고객 생애 가치)** 를 높이는 리텐션 중심의 전략으로의 전환이 필수적임을 도출함.

### 이탈 위험군(25.2%)의 정밀 진단
- 전체 고객의 25.2%에 달하는 이탈 위험군은 특히 수도권 외곽 지역에서 높은 이탈률을 보임. 이는 해당 지역의 배송 만족도나 서비스 점검이 필요함을 시사함.
- 카테고리 편중 현상: 이탈 위험군의 60% 이상이 '사무용품'에만 치중된 구매 패턴을 보임. 이를 해결하기 위해 가구/가전 등 고단가 카테고리의 20% 할인 쿠폰을 제공하여 구매 저변을 확대하는 전략 제안.

### 일반·우수 고객 타겟 전략
- 데이터 기반 자동화: 우수 고객의 평균 재구매 주기인 84일을 기반으로, 고객 이탈 방지의 골든타임인 **'구매 후 80일 차'** 에 재구매 리마인드 알림을 발송하는 CRM 자동화 시나리오 설계.
- 수익성 강화: 일반 고객에게 우수 고객의 주 구매 품목인 '책장/가구' 등을 **교차 판매(Cross-selling)** 하여 우수 고객 패턴으로 유도하고, 우수 고객에게는 신상품 얼리버드권 등 차별화된 VIP 케어 제공.
